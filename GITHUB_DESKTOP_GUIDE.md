# 使用 GitHub Desktop 推送專案

## 📋 步驟說明

### 步驟 1: 開啟 GitHub Desktop

1. 開啟 GitHub Desktop 應用程式
2. 如果還沒登入，請先登入你的 GitHub 帳號

### 步驟 2: 加入本地 Repository

1. 在 GitHub Desktop 中，點擊左上角的 **File** 選單
2. 選擇 **Add Local Repository...**
3. 在彈出的視窗中，點擊 **Choose...** 按鈕
4. 選擇 `~/Desktop/happyschool` 資料夾
5. 點擊 **Add repository**

### 步驟 3: 檢查檔案狀態

- 你應該會看到所有檔案都在 "Changes" 標籤中
- 如果看到 "No local changes"，表示檔案已經在 Git 中了（這是正常的）

### 步驟 4: 發布到 GitHub

1. 在 GitHub Desktop 底部，點擊 **Publish repository** 按鈕
   - 如果沒有看到這個按鈕，點擊右上角的 **Publish repository** 按鈕
2. 在彈出的視窗中：
   - **Name**: 輸入 repository 名稱，例如：`sdgs-tech-showcase`
   - **Description** (可選): 輸入描述，例如：`SDGs 科技素養專題成果展`
   - **Keep this code private**: **不要勾選**（GitHub Pages 免費版需要 Public）
3. 點擊 **Publish repository**

### 步驟 5: 等待上傳完成

- GitHub Desktop 會顯示上傳進度
- 完成後，你的專案就會出現在 GitHub 上了

### 步驟 6: 啟用 GitHub Pages

1. 在 GitHub Desktop 中，點擊 **Repository** > **View on GitHub**
   - 或直接在瀏覽器開啟：`https://github.com/<你的帳號>/<repository名稱>`
2. 點擊 **Settings**（設定）
3. 在左側選單找到 **Pages**
4. 在 **Source** 區塊：
   - 選擇 **Deploy from a branch**
   - **Branch** 選擇 **main**
   - **Folder** 選擇 **/ (root)**
5. 點擊 **Save**
6. 等待 1-2 分鐘，GitHub 會顯示你的網站網址：
   - `https://<你的帳號>.github.io/<repository名稱>/`

## ✅ 完成！

你的網站現在應該已經上線了！

## 🔍 測試網站

1. 開啟網站網址
2. 測試功能：
   - ✅ 語言切換
   - ✅ 作品卡片顯示
   - ✅ SDG 篩選
   - ✅ 點擊作品連結

## 📱 產生 QR Code

1. 使用 QR Code 產生器（如 https://www.qr-code-generator.com/）
2. 輸入你的網站網址
3. 下載並列印 QR Code
